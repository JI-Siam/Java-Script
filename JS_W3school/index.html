Switch Statement also similar to c++

let x=20 ; 
          switch (x){
            case 20 : 
                document.write('Adolscent');
                break;
            case 21 : 
                document.write('Adult');
                break; 
            default : 
                document.write('Baby');
          }

This is result is derived using switch case
